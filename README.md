Descripción

Sistema web para la gestión de pedidos en un negocio local, permitiendo registrar, visualizar y administrar órdenes de manera eficiente.

Problema identificado

El negocio gestiona pedidos manualmente mediante WhatsApp, lo cual genera:

Pérdida de pedidos
Errores en órdenes
Desorganización en horas pico
Solución

Se desarrolló una aplicación web que permite:

Registrar pedidos estructurados
Visualizar órdenes en tiempo real
Mantener un historial
Arquitectura

Cliente → Frontend → Backend → Base de datos
